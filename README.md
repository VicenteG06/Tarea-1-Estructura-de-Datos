# Tarea-1-Estructura-de-Datos

## Código Tarea 1:
Para ejecutar el código primero debes compilarlo en la carpeta raíz.

````
gcc tdas/*.c tarea1.c -Wno-unused-result -o tarea1
````

Y luego ejecutar:

````
./tarea1
````

Probar hasta la opción "Mostrar Categorías", lo demás aún no está terminado. 🥺
